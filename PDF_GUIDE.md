# 📄 راهنمای تبدیل MD به PDF (استایل GitHub)

## 🎯 روش‌های موجود

### 🥇 **روش 1: Browser-based (توصیه شده)**
```bash
python browser_to_pdf.py
```

**مزایا:**
- ✅ نیاز به نصب اضافی ندارد
- ✅ کیفیت عالی مشابه GitHub
- ✅ فونت Vazir برای فارسی
- ✅ کنترل کامل روی فرمت

**مراحل:**
1. اسکریپت رو اجرا کن
2. فایل HTML در مرورگر باز میشه
3. `Ctrl+P` فشار بده
4. "Save as PDF" انتخاب کن
5. تنظیمات: A4, margins: normal
6. Save کن

### 🥈 **روش 2: wkhtmltopdf (خودکار)**
```bash
# ابتدا wkhtmltopdf نصب کن:
winget install wkhtmltopdf

# سپس اجرا کن:
python md_to_pdf.py
```

**مزایا:**
- ✅ کاملاً خودکار
- ✅ کیفیت بالا
- ✅ تنظیمات قابل سفارشی‌سازی

### 🥉 **روش 3: GitHub Actions (آنلاین)**
1. به تب **Actions** برو
2. **Convert MD to PDF** رو اجرا کن
3. فایل‌های PDF از **Artifacts** دانلود کن

## 🎨 ویژگی‌های استایل GitHub

### **عناوین:**
- H1: 2em با خط زیر
- H2: 1.5em با خط زیر  
- H3: 1.25em
- H4: 1em

### **رنگ‌ها:**
- متن: #24292f (خاکستری تیره)
- عناوین: #1f2328 (مشکی)
- نقل قول: #656d76 (خاکستری)
- پس‌زمینه کد: #f6f8fa (خاکستری روشن)

### **فونت‌ها:**
- متن: Vazir, Segoe UI
- کد: SFMono-Regular, Consolas

### **فاصله‌گذاری:**
- عناوین: 24px بالا، 16px پایین
- پاراگراف‌ها: 16px پایین
- صفحه: 45px padding

## 📋 مثال نتیجه

فایل MD شما:
```markdown
## عنوان

متن عادی با **bold** و `code`

> نقل قول

- لیست آیتم
```

تبدیل میشه به PDF با:
- ✅ فونت Vazir برای متن فارسی
- ✅ فرمت‌بندی مشابه GitHub
- ✅ رنگ‌بندی و فاصله‌گذاری مناسب
- ✅ تراز راست برای RTL

1. **بهترین کیفیت**: روش Browser-based
2. **سریع‌ترین**: wkhtmltopdf (اگه نصب باشه)
3. **بدون نصب**: GitHub Actions
4. **فونت فارسی**: همه روش‌ها پشتیبانی می‌کنن
5. **تنظیمات print**: A4, margins normal, background graphics on
