**بهترین روش برای درک نمودار بالا، حرکت از چپ به راست است:**

*سرویس ۱ تا N:*  
این بخش نشان‌دهنده سرویس‌های مختلفی است که از طریق APIهای ارائه‌شده توسط سرورهای اعلان، نوتیفیکیشن ارسال می‌کنند.

*سرورهای اعلان:*  
این سرورها قابلیت‌های زیر را ارائه می‌دهند:  
• ارائه API برای ارسال نوتیفیکیشن توسط سرویس‌ها. این APIها فقط به صورت داخلی یا برای کلاینت‌های تأییدشده قابل دسترسی هستند تا از اسپم جلوگیری شود.  
• انجام اعتبارسنجی‌های اولیه مانند بررسی صحت ایمیل، شماره تلفن و غیره.  
• پرس‌وجو از پایگاه داده یا کش برای دریافت اطلاعات مورد نیاز جهت نمایش نوتیفیکیشن.  
• قراردادن داده‌های نوتیفیکیشن در صف‌های پیام برای پردازش موازی.  

---
**مثالی از API ارسال ایمیل:**  
`POST https://api.example.com/v/sms/send`
